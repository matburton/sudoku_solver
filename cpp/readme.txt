
To build this project:

1. Download boost and extract it into libraries/boost/src (it was too big to include)
   http://www.boost.org/users/download
   
2. If you're using VS2012 open build/windows/vs2012/sudoku_solver.sln, you're done

3. Otherwise download premake4 http://industriousone.com/premake/download

4. Issue a command like 'premake4 gmake' in build/ to create everything for your IDE/platform
   http://industriousone.com/premake/download
